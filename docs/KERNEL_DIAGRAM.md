# WHAOS Kernel Architecture Diagram

## Boot Flow

```
┌─────────────────────────────────────────────────────────────────────┐
│                        BIOS / UEFI Firmware                          │
└─────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────┐
│  GRUB2 Bootloader                                                    │
│  - Loads vmlinuz-whacos, initramfs                                   │
│  - Parses kernel cmdline (root=, init=)                              │
└─────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────┐
│  WHAOS Kernel (Linux-based)                                          │
│  - x86-64 entry: arch/x86/boot/head_64.S                             │
│  - start_kernel() → rest_init() → kernel_init()                      │
└─────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────┐
│  Userspace Init: whad (PID 1)                                        │
│  - Mounts /proc, /sys, /dev                                          │
│  - Runs /etc/whad/rc.conf                                            │
│  - Starts getty, X11, display manager                                │
└─────────────────────────────────────────────────────────────────────┘
```

## Kernel Layer Overview

```
┌──────────────────────────────────────────────────────────────────────┐
│                     User Applications (WHAOS Desktop)                  │
└──────────────────────────────────────────────────────────────────────┘
                                    │
                                    │  syscalls
                                    ▼
┌──────────────────────────────────────────────────────────────────────┐
│  System Call Interface (Linux ABI)                                    │
│  open, read, write, fork, execve, mmap, ...                           │
└──────────────────────────────────────────────────────────────────────┘
                                    │
        ┌───────────────────────────┼───────────────────────────┐
        ▼                           ▼                           ▼
┌───────────────┐         ┌───────────────┐         ┌───────────────┐
│  VFS (ext4,   │         │  Process      │         │  Memory       │
│  procfs, etc) │         │  Scheduler    │         │  (paging,     │
│               │         │  fork/exec    │         │   mmap)       │
└───────────────┘         └───────────────┘         └───────────────┘
        │                           │                           │
        └───────────────────────────┼───────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────┐
│  Drivers (VirtualBox devices)                                         │
│  - Block: IDE, AHCI (SATA)                                            │
│  - Display: VMSVGA (drm_vmwgfx), VESA fallback                        │
│  - Input: PS/2 keyboard, PS/2 mouse                                   │
│  - Network: Intel e1000 (Intel PRO/1000)                              │
└──────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────┐
│  Hardware (VirtualBox emulated devices)                               │
└──────────────────────────────────────────────────────────────────────┘
```
