# WHAOS Core Applications

All applications use **WHAOS** branding in titles, icons, and UI.

## Application Specifications

### 1. WHAOS File Explorer

- **Purpose**: File manager (browse, copy, move, delete, create)
- **Base**: PCManFM, Thunar, or Nautilus (customized)
- **Features**: Two-pane optional, icon/list view, address bar, navigation
- **Branding**: Window title "WHAOS File Explorer", WHAOS icon

### 2. WHAOS Control Center

- **Purpose**: System settings (appearance, themes, display, user accounts)
- **Base**: Custom GTK app or LXControl
- **Sections**: Appearance, Display, Users, Network, Date/Time
- **Branding**: "WHAOS Control Center"

### 3. WHAOS Text Editor

- **Purpose**: Plain text editing
- **Base**: gedit, Geany, or Leafpad
- **Features**: Open/save, syntax highlighting, find/replace
- **Branding**: "WHAOS Text Editor"

### 4. WHAOS Terminal

- **Purpose**: Command-line shell
- **Base**: GNOME Terminal, lxterminal, or xterm
- **Features**: Tabs, copy/paste, scrollback
- **Branding**: "WHAOS Terminal"

### 5. WHAOS Task Monitor

- **Purpose**: Process viewer, resource monitor
- **Base**: GNOME System Monitor, htop
- **Features**: Process list, CPU/RAM usage, kill process
- **Branding**: "WHAOS Task Monitor"

## Desktop Integration

- **Start Menu** entries for all core apps
- **Desktop shortcuts** (optional)
- **MIME types** – File Explorer opens folders, Text Editor opens .txt
